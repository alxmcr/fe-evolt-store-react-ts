
# eVolt

An e-commerce app where you can buy technology products.


## Run Locally

Clone the project

```bash
  git clone https://github.com/alxmcr/fe-evolt-store-react-ts
```

Go to the project directory

```bash
  cd fe-evolt-store-react-ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
